### Objetivo:
Projeto ensina como construir um projeto do zero (Parking Control API), com as 5 operações básicas:
- Buscar todos 
- Buscar por id
- Criar um novo recurso
- Alterar um recurso
- Delete um recurso
 

### Tecnologias utilizadas:
- JDK 11 
- Spring 
- Maven

### Dependências:
- Spring Web
- Spring Data JPA
- Spring Validation
- Lombok
- PostgreSQL
